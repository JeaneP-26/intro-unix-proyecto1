##------------------**First Term Final Project**---------------------


This project, which we developed together with my teammates Francis Acuña, Edison 
Yambay, and my person Jeanella Paredes aims to apply in a practical way several
key concepts from the course, especially realted to UNIX/Linux and cybersecurity.

Throughout the different sections, we focus on learning objectives such as understanding
how UNIX/Linux works and how it differs from other operating systems, becoming familiar
with its file system, permissions, and terminal commands, and using tools like GPG to
encrypt, decrypt, and sign information. We also strengthen our knowledge of security
concepts, incluiding disk encryption, password management, hash functions, and repository management using Git


Overall, thus project allows us not only to understand the theory, but also to apply it in real scenarios,
strengthening both our technical skills and our awareness of the importance of security in the digital environment.

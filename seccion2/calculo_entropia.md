MANUAL CALCULATION OF THE ENTROPY:

Manual calculation of the entropy of 5 different passwords using Shannon's formula: 

H = L × log2(N)

1. Calculation 1: Only Lowercase Letters 
Password: sistemasoperativos
Length (L): 18
Alphabet Size (N): 26 (a-z)

-------Step-by-step--------------------:
H= 18×log_2⁡(26)
H=18×4.7004 
**Result**: H=84.61 Bits


2. Calculation 2: Alphanumeric Password
Password: Debian13Server2026
Length (L): 18
Alphabet Size (N): 62 (26 lowercase + 26 uppercase + 10 digits)


-------Step-by-step---------------------:
H=18×log_2⁡(62)
H=18×5.9542 
**Result**: H=107.17 bits

3. Calculation 3: Complex Password with Symbols
Password: P@ssw0rd_Luks!26
Length (L): 16
Alphabet Size N): 94 


-------Step-by-step----------------------:
H=16×log_2⁡(94)
H=16×6.5546 
**Result**: H=104.87 bits
